# Intro to R teaching materials

## What this repo contains


### 📁 Data Folder
Contains the following **synthetically derived datasets**:
- `patients.csv`
- `admissions.csv`

### 📁 Scripts Folder
Contains two scripts:
1. **Script 1** – Introduction to setting up and importing data  
2. **Script 2** – Basic data science work cycle

## How to download
Click the green **Code** button, then **Download ZIP**.

## How to use
1. Download and unzip
2. Import datasets from data folder
3. Import scripts from script folder
